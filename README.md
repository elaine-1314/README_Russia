## 1.Введение

BLIKVM — это прикладная плата CM4 для [PiKVM](https://pikvm.org/). Основные функции этого продукта включают захват видео, адаптер ATX, USB 3.0, OTG, Gigabit Ethernet, OLED и RTC. Продукт имеет индивидуальную металлическую оболочку для отвода тепла и защиты печатной платы. Это видео с демонстрацией возможностей.[video](https://www.youtube.com/watch?v=d7I9l5yG5M8).В настоящее время продукт полностью совместим с[**v3-hdmi-rpi4-latest.img**](https://files.pikvm.org/images/v3-hdmi-rpi4-latest.img.xz).

## 2.Features

| --------------------------------------------- | ------------------------------------------ |
| <img src="/images/docs_image/2-1.png" alt="drawing"/> | <img src="/images/docs_image/2-2.png" alt="drawing"/> |

Here is a brief introduction. For details, please click <a href="https://github.com/ThomasVon2021/pikvm-CM4-Board/wiki#features" target="_blank">WiKi</a>
1. **HDMI IN** port with I2S
2. **ATX** controller interface (power on/off, reboot control, PWR and HDD ACT LEDs)
3. **USB3.0 port**  x 2
4. **USB-C OTG**
5.**Real Time Clock (RTC)**
6.**Gigabit Ethernet**
7.**Activity** LED 
8.**Micro SD Card** socket
9.**Power** LED
10. **I2C** display connector
11. **nRPI_BOOT** Jumper
12.**USB-C power input** 
13. **FAN** connector 5V
14. **CSI-2 date lanes switch**
15. CM4 Module Connectors
